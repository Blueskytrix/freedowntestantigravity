# 06 - Limitaciones

## Limitaciones Principales

1. **Stack fijo:** Solo React + Vite + Tailwind
2. **Sin backend directo:** Solo via Supabase Edge Functions
3. **Archivos read-only:** No puedo modificar package.json, tsconfig.json
4. **Sin memoria entre sesiones:** Olvido todo al cerrar
5. **Context limitado:** Solo ~40K tokens usados de 200K disponibles
6. **20 herramientas:** vs 50+ necesarias para libertad total
7. **Sin control de modo:** Lovable decide si puedo escribir código
8. **Sistema de créditos:** No lo veo pero limita al usuario
